Formatting

About this case converter

Quickly convert text between different capitalization styles. This is useful for headlines, titles, social media posts and fixing accidental caps lock mistakes.

All conversions happen instantly in your browser and work even when you are offline after the page is loaded.

FAQ
  • What is title case? Title case capitalizes the first letter of most words, excluding some short words depending on style.
  • What is sentence case? Sentence case capitalizes only the first letter of the first word and proper nouns.
  • Will this change my punctuation? No, it only changes letter casing.

Base64 Encoder / Decoder

Convert text to Base64 and decode Base64 back to text, fully in your browser.

Developer

About this Base64 tool

Base64 is a way of representing binary data using text characters. Developers often use it in data URLs, simple encoding or when moving data between systems that expect text.

This tool uses built-in browser functions to encode and decode Base64 locally. It's suitable for most everyday use cases, but not for strong encryption or security-critical workflows.

About Text Tool Hub

Text Tool Hub is a collection of lightweight, browser-based tools for writers, students, developers and content marketers. Every tool runs entirely in your browser: we don't store, log or transmit your text or code to any server.

The goal is to help you get small tasks done quickly – counting words, cleaning up text, formatting JSON, encoding/decoding data or running quick percentage calculations – without installing software or creating an account.

As the site grows, we'll keep adding focused, single-purpose tools with clear interfaces and zero clutter.

FAQ

Yes. The plan is to keep all tools free to use and support the site only with privacy-friendly ads.

No. All calculations and processing happen in your browser. We don't store your inputs on a server.

Yes. The layout is responsive and designed to work on phones, tablets and desktops.
